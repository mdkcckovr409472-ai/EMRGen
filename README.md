# Artifact for the paper "EMRGen: Automated Generation of Executable Metamorphic Relations via LLM-Aided Program Invariants Exploration"

This repository contains data and code for the paper "EMRGen: Automated Generation of Executable Metamorphic Relations via LLM-Aided Program Invariants Exploration"

## Code 

* Code of the EMRGen framework, containing all the scripts and programs required to replicate our project and experiments.

## Data

* All data used and generated during the implementation, including the benchmark subject programs and experimental datasets.

Please refer to [data.md](data/data-info.md) for more details.

And please refer to [requirement.txt](requirement.txt) for istallation requirements.
